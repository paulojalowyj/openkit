# Sprint Goal

Entregar um sistema de upgrade seguro para a CLI do OpenKit:

- `openkit upgrade` atualiza `.opencode/` e `opencode.json` com deteccao de customizacoes via manifest.
- Fluxos interativo e nao-interativo claros, com defaults seguros em CI.
- Testes cobrindo o algoritmo de merge/diff e os principais cenarios.
